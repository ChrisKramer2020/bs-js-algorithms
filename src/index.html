A Palindrome is a word or sentence that is spelled the same way forward as it is backwards.
Ex: Bob

Note You'll need to remove all non-alphanumeric characters (punctuation, spaces and symbols) and turn everything lower case in order to check for palindromes.

Solution

Function palindrome(str) {
return str.replace(/[\W_]/g, '').toLowerCase() === str.replace(/[\W_]/g, '').toLowerCase().split('').reverse().join('');
}

A factorial function (symbol !) means to multiply a series of descending numbers
Ex: 4! means 4 * 3 * 2 * 1 = 24

Solution

function factorialize(num) { if (num === 0) { return 1; } return num * factorialize(num-1); } factorialize(5);

Solution

Function reverse(str) { return str.split("").reverse().join(""); } reverse("Hello")

find longest word in string

Solution

function findLongestWord(str) { var words = str.split(' '); var maxLength = 0; for (var i = 0; i < words.length; i++) { if (words[i].length > maxLength) { maxLength = words[i].length; } } return maxLength; }
